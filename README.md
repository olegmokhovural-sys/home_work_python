# home_work_python
Репо для домашки
## Описание
Проект создан для выполнения домашней работы Skypro по созданию программы
управления банковским приложением.
Приложение создается в процессе работы над рядом домашних заданий.
## Установка
1. Клонируйте репозиторий '''https://github.com/username/home_work_python.git'''
2. Создайте виртуальное окружение (рекомендуется):

python -m venv venv
source venv/bin/activate  # для Linux/Mac
#### или
venv\Scripts\activate  # для Window
3. Установите необходимые зависимости:

pip install -r requirements.txt



## Использование
1. Файл 'widget.py' выполняет функцию маскировки номера счета и карты.
Принимает на вход номера карт и счетов, выдает замаскированный номер
счета и карты.

** # Пример для карты
Visa Platinum 7000792289606361  # входной аргумент
Visa Platinum 7000 79** **** 6361  # выход функции

** # Пример для счета
Счет 73654108430135874305  # входной аргумент
Счет **4305  # выход функции

2. Файл 'processing.py' выполняет функцию сортировки словарей по
ключу 'статус' и дате

*** # Выход функции со статусом по умолчанию 'EXECUTED'
[{'id': 414288290, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, 
 {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}]

*** # Выход функции, если вторым аргументом передано 'CANCELED'
[{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, 
 {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}]

*** # Выход функции сортировки по дате
[{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 615064591, 'state': 'CANCELED', 'date': '2018-10-14T08:21:33.419441'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, {'id': 939719570, 'state': 'EXECUTED', 'date': '2018-06-30T02:08:58.425572'}]

3. Файл 'generators.py' содержит 3 функции

*** Функция filter_by_currency фильтрует по валюте операции. Пример выхда функции, отфильтрованной по валюте 'USD':
{         "id": 939719570,
          "state": "EXECUTED",
          "date": "2018-06-30T02:08:58.425572",
          "operationAmount": {
              "amount": "9824.07",
              "currency": {
                  "name": "USD",
                  "code": "USD"
              }

***Функция 'transaction_descriptions' фильтрует описания проведенных операций. Пример выхода функции:
Перевод организации
    Перевод со счета на счет
    Перевод со счета на счет

*** Функция 'card_number_generator' генерирует номера карт в заданном диапазоне значений. Пример выхода функции в диапазоне 1-3:
0000 0000 0000 0001
    0000 0000 0000 0002
    0000 0000 0000 0003
